# DiscordStatus 🌵
Requirements:
🍁 Python

🎉 Library **pypresence**

# How to install?
```sh
git clone https://github.com/Kasztan566/DiscordStatus/
```
# How to configure?
Open **config.json**
```
{
    "name": "Template status",
    "description": "https://github.com/Kasztan566/",
    "image": "<image name>",
    "id": "<Enter your discord application id>"
}
```
